# odin-recipes

This is the first project from The Odin Project's Foundations course.

The goal of this project is to practice the basic HTML concepts covered in the course, including creating pages, using links, adding images, and structuring content with HTML elements.

Recipe references used in this project:

- James Delmage's "Creamy Chicken Stroganoff" from https://www.sipandfeast.com/chicken-stroganoff/
- The "Viennese Apple Strudel" recipe from https://www.wien.info/en/dine-drink/viennese-cuisine/recipes/sweets-desserts/apple-strudel-339140
- The "Opera Cake" recipe from https://www.meilleurduchef.com/en/recipe/opera-cake.html
